
Aira Cheska Villamor Portfolio

A modern, responsive portfolio website built with Python Streamlit showcasing the skills, projects, and contact information of Aira Cheska Villamor.

✨ Features
Home Page: Hero section with introduction and quick navigation
About Page: Detailed bio, skills, and experience
Contact Page: Form and contact information
Responsive Design: Works perfectly on desktop, tablet, and mobile
Fast Loading: Optimized Streamlit performance
Professional UI: Clean, modern design with smooth animations
🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
Styling: Streamlit custom CSS
Deployment: Streamlit Cloud / Railway / Heroku
🚀 Quick Start
Prerequisites
Python 3.8+
Git
Installation
Clone the repository
bash


git clone https://github.com/yourusername/aira-cheska-portfolio.git
cd aira-cheska-portfolio
Create virtual environment
bash


python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
Install dependencies
bash


pip install -r requirements.txt
Run the app
bash


streamlit run app.py
The app will open at http://localhost:8501

Copy code
